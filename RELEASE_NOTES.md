## 🎯 主要功能

### 核心翻译功能
- ✅ **英汉互译**: 支持英→中、中→英、自动检测
- ✅ **划词翻译**: `Ctrl+Space` 快捷键激活
- ✅ **悬浮窗**: 快速查看翻译结果
- ✅ **朗读功能**: 支持中英文朗读

### 高级功能
- ✅ **截图 OCR**: 识别屏幕英文并翻译
- ✅ **剪贴板监听**: 自动检测和翻译
- ✅ **生词本**: 本地离线词库管理
- ✅ **离线能力**: 所有功能本地运行，无需网络

## 📋 系统要求

- **操作系统**: Windows 7 或更高版本
- **处理器**: Intel Core i3 或同等
- **内存**: 最少 2GB RAM
- **磁盘**: 最少 500MB 可用空间
- **网络**: 可选（所有功能可离线使用）

## 🚀 安装方式

### 方式 1: 标准安装程序（推荐）
```
1. 双击 Translate.Desk-0.4.2-win-x64.exe
2. 按照向导完成安装
3. 选择安装目录（可自定义）
4. 完成后自动启动应用
```

### 方式 2: 自定义安装位置
在 NSIS 安装程序中选择"自定义安装路径"选项

### 方式 3: 卸载
- Windows 设置 → 应用 → 搜索"Translate Desk" → 卸载
- 或通过安装程序的卸载选项

## 🔍 验证安装

安装完成后，可以进行以下验证：

1. **启动应用**
   - 桌面快捷方式或开始菜单中找到 Translate Desk
   - 点击启动应用

2. **测试基础功能**
   ```
   - 输入 "supplier" → 应显示 "供应商"
   - 输入 "合同" → 应显示 "contract"
   - 点击朗读按钮 → 听到英文朗读
   ```

3. **测试快捷键**
   - 在任何窗口选中英文文本
   - 按 `Ctrl+Space` → 应弹出浮窗显示翻译

4. **测试 OCR**
   - 点击"截图 OCR" 按钮
   - 选择屏幕区域
   - 应自动识别并翻译
